Our solution implements a lightweight diffusion model tailored to quark–gluon jet images. The dataset provides 3‑channel calorimeter representations (ECAL, HCAL, Tracks) of size 125×125, which we process in PyTorch as images. We define a simple CNN backbone that preserves spatial resolution and train it on a small subset of the data, where the forward diffusion step adds Gaussian noise and the network learns to predict that noise. This setup demonstrates the core principle of diffusion models—learning to reverse a noising process—without requiring full dataset training or heavy compute. For clarity and reproducibility and because few samples were used due to computational requirements, we focused on the training pipeline itself and did not include image plotting in the final implementation.
