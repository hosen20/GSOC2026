## 🎯 Why I applied to ML4Sci
I applied to ML4Sci because I want to contribute machine learning and QML solutions that directly accelerate scientific discovery. ML4Sci brings together domain experts and ML practitioners to solve high‑impact problems. My motivation is to work at the intersection of particle physics and modern ML, where improvements in algorithms and representations can meaningfully reduce experimental costs and speed up analysis workflows.

**Key personal motivations**
- **Make ML useful for science** by producing reproducible, well‑documented code that domain scientists can run and extend.  
- **Bridge theory and practice** by turning physics concepts (symmetries, conservation laws) into practical ML modules.  
- **Learn and contribute** to quantum and physics‑aware ML techniques while sharing results openly.

This repository and the proposed work align with ML4Sci priorities in two concrete ways:

- **Scientific impact** — discovering symmetries in detector representations helps build models that generalize better and require fewer labeled examples, directly supporting efficient analysis of HL‑LHC scale data.  
- **Cross‑disciplinary innovation** — the project combines physics knowledge, semi‑supervised learning, and equivariant modeling; the companion Quantum RL proposal explores quantum methods for compute and sample efficiency, showing commitment to exploring multiple promising directions for ML4Sci.
